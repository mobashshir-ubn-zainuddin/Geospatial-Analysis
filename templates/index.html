Remote Sensing for Mineral Exploration

Project Overview

This web application demonstrates the application of hyperspectral remote sensing for mineral exploration in the Cuprite Nevada area. The workflow showcases the process of loading, processing, and analyzing hyperspectral satellite imagery to detect and map the presence of kaolinite, a clay mineral often associated with hydrothermal alteration zones.

Each stage of the workflow can be explored in detail through the navigation menu. To run the complete workflow and generate results, use the "Run Simulation" button in the top right corner.

About This Demo

This demonstration showcases skills relevant to mineral exploration, space science, and geospatial analytics. The techniques used include:

  • Hyperspectral image processing
  • Spectral analysis and mineral identification
  • Geospatial data processing and visualization
  • Python-based scientific computing for remote sensing

The codebase demonstrates the practical application of these techniques to solve real-world problems in mineral exploration and geological mapping.
